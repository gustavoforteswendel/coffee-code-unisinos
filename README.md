# ☕ CoffeeCode - Cardápio Digital

Projeto desenvolvido para a disciplina de **Desenvolvimento Web - Unisinos**.

O sistema simula um **cardápio digital de cafeteria**, permitindo que o usuário visualize produtos, selecione opções, adicione ao carrinho e finalize um pedido.

---

# 📚 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias básicas de desenvolvimento web:

- **HTML5** → estrutura das páginas
- **CSS3** → estilização e layout
- **JavaScript (Vanilla JS)** → lógica da aplicação
- **LocalStorage** → armazenamento temporário do carrinho

---

# ⚙️ Funcionalidades

O sistema possui as seguintes funcionalidades:

- Visualização do **cardápio de cafés**
- Página individual de **produto**
- Seleção de:
  - tamanho do café
  - tipo de leite
- Adição de produtos ao **carrinho**
- Controle de **quantidade de itens**
- Página de **resumo do pedido**
- Página de **confirmação de pedido**
